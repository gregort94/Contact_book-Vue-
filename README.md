# contact_book
> A Vue.js project
## Project description
This SPA is a contact book where the following features are implemented:
* Adaptive for different screen sizes
* Get contacts data from server (mo—Åk)
* First letter grouping and sorting
* Search engine
* Edit and remove contacts
